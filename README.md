# Genshin Impact Character Build Resizer

A showcase for Genshin Impact character build resizer from enka.network (landscape) to portrait mode (16:9). Perfect for Stories, Reels, TikTok, or YouTube Shorts!

## Features
- Single Mode
- Lineup Mode
- No Internet Needed just run `index.html` on browser.

## How to Use
1. Upload image by clicking the upload zone or dragging and dropping the image.
2. Image will be resized to portrait mode (16:9).
3. Download image by clicking the download button.
